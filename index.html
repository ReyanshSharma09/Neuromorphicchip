💡 How Parameters Work

Adjust these to change how the chip processes your input. Each parameter affects neuron behavior and output accuracy.

Neuron Model

LIF: Neurons accumulate input and leak over time. Most common model - balances biology and efficiency.

Firing Threshold 1.0

Effect: Higher = neurons need more input to fire. Lower = more sensitive, more spikes generated.

Leak Rate (τ) 0.95

Effect: How fast voltage decays. Higher = longer memory. Lower = faster forgetting, needs constant input.

Time Steps 50

Effect: Processing duration. More steps = more accurate but slower. Fewer = faster but may miss patterns.

Hidden Neurons 16

Effect: More neurons = better pattern recognition. Fewer = faster, less energy, simpler patterns.

Spike Encoding

Rate: Input intensity = spike frequency. Simple and robust for most tasks.

Noise Level 5%

Effect: Simulates real-world signal noise. Higher = more realistic but harder classification.

🔄 Processing Flow
1. Encode: Input → Spike train
2. Propagate: Spikes flow through layers
3. Integrate: Neurons accumulate charge
4. Fire: Threshold reached → spike!
5. Decode: Count output spikes → class
⚡ Why Spikes?

Unlike traditional AI that computes continuously, neuromorphic chips only compute when spikes occur. This event-driven approach uses 10-100x less energy - perfect for edge devices and real-time sensing.
